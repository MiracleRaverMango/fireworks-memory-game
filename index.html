🎇 Fireworks Memory Game

Welcome! Here’s how to play:

  • 💡 Watch the fireworks explode in sequence.
  • 🖱 Click the fireworks back in the same order.
  • 🏆 Survive 10 rounds to trigger the Grand Finale.
  • 🔥 Endless Mode continues ramping speed after round 10.

Controls:
A = Change difficulty
M = Mute/unmute sound
E = Endless mode
D = Daily challenge
I = Show/Hide instructions

  • Mode: ⚙️ (☰ → Difficulty )
  • Sound: 🔊/🔇 (☰ → Sound)
  • Replay: ⟲ (☰ → Replay)
  • Endless: ∞ (☰ → Endless Mode)
  • Daily: 📅 (☰ → Daily Challenge)
  • Info: ❓ (☰ → Instructions)
